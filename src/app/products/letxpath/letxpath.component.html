LetXPath: Your Ultimate XPath & CSS Selector Finder

An open-source tool to find XPath and CSS selectors effortlessly.

How to Use LetXPath? 🤔

  1. Install the Extension: Restart your browser after installation.
  2. Navigate to the Desired Page: Open the page from which you need XPath.
  3. Open DevTools: Click inspect or press F12.
  4. Access LetXPath: Open it from the sidebar panel.
  5. Get Your XPath: Click the element to obtain its XPath.

Features 💭

  • Single-Click XPath: Obtain XPath instantly.
  • Smart XPath Generation: Direct element and parent-child relations.
  • Dynamic XPaths: Supports `following`, `preceding`, etc.
  • Axis-Based XPaths: Generates XPaths based on axes.
  • User-Enhanced Axes: Allows user-customized axis selection.

Code Snippets 🎓

Supports Selenium (Java, Python, C#), Protractor JS, and Playwright (Node & Java).

Why Choose LetXPath? 🤔

Unlike other XPath tools, LetXPath provides XPaths, snippets, and video tutorials on building the tool itself.
